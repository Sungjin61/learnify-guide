05 / FAQ

이런 경우에는요.

사용 중 자주 만나는 상황과 대처법.

Udemy 계정은 어떻게 얻을 수 있나요? ＋

Udemy 계정은 쏘칼패스 신청 시 제공받을 수 있습니다. 쏘칼패스는 분기별로 신청이 진행되며, 자세한 내용은 아래 링크에서 확인해주세요.

  • →신청안내 페이지 바로가기
  • →2026년 교육일정 페이지 바로가기
「학습 컨텐츠 DB 검색 결과, [주제] 과정은 찾을 수 없습니다.」 라고 나와요. ＋

해당 주제와 관련된 학습 과정이 현재 데이터베이스에 등록되어 있지 않은 경우입니다.

  • →키워드를 바꿔서 다시 검색해보세요 (예: 「양자 컴퓨팅」 → 「Quantum Computing」)
  • →더 넓은 범위의 주제로 시도해보세요
  • →영문 키워드로도 시도 (Udemy 영어 강의가 많습니다)
「학습 데이터 DB에 요청하신 데이터가 없습니다」 + 범위 구조도가 표시돼요. ＋

지식검색에서 관련 문서를 찾지 못한 경우입니다.

  • →함께 표시된 검색 범위 구조도에서 주제가 범위 안에 있는지 확인하세요
  • →범위 내 주제로 질문을 다시 작성해보세요
  • →HR 급여·복리후생·일반 상식은 Learnify 검색 범위가 아닙니다
여러 프로그램 정보가 한꺼번에 나오거나, 어떤 걸 원하는지 되물어봐요. ＋

「역량인증 일정 알려줘」처럼 여러 프로그램 공통 키워드를 사용했을 때 발생합니다.

  • →프로그램명을 구체적으로 명시하세요: 「SW프로그래밍 역량인증시험 일정 알려줘」
  • →나열된 항목 중 원하는 것을 선택해 추가 질문하세요
업무와 관련 없는 질문(날씨·뉴스 등)을 하면 안내 메시지가 나와요. ＋

Learnify는 SW College 교육 관련 질문에 특화된 챗봇입니다. 일반 상식·날씨·뉴스 등에는 답변할 수 없습니다.

대신 이런 질문을 시도해보세요 — 「AI Specialist에 대해 알려줘」 / 「Python 학습 과정 추천해줘」 / 「데이터 엔지니어링 학습 로드맵을 만들어줘」

한 번에 두 가지를 물어보고 싶어요 (소개 + 강의 추천). ＋

현재 Learnify는 한 번에 하나의 검색 의도를 처리합니다. 두 가지 이상이 섞이면 정확도가 떨어집니다.

  • 1먼저 지식검색: 「AI Specialist에 대해 알려줘」
  • 2답변 확인 후: 「AI 관련 학습 과정 추천해줘」
  • 3필요시: 「AI/ML 학습 로드맵을 만들어줘」
추천받은 과정의 링크가 열리지 않아요. ＋

링크 오류가 지속되면 SW College 담당자에게 문의해주세요.

로드맵 추천에서 일부 난이도만 나와요. ＋

정상 동작입니다. Learnify는 해당 주제에 실제 존재하는 난이도의 과정만 추천합니다.
